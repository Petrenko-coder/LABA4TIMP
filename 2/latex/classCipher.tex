\doxysubsection{Подробное описание}
Описание класса \mbox{\hyperlink{classCipher}{Cipher}}. 

Ключ устанавливается в конструкторе, Для зашифрования и расшифрования предназначены методы encrypt и decrypt. \begin{DoxyWarning}{Предупреждения}
Реализация производится только для русского языка. С использованием wstring. 
\end{DoxyWarning}
